Trash Belt

2019-02-10

Fly through space and try not to crash! A game for Android where your goal is to collect coins while avoiding space trash.

I used Unity a lot for prototyping smaller games, but always wanted to try Godot. This game is the result of me playing around with it over the course of a weekend and adding some more features over time.

Update in 2024:

The original PlayStore page is now offline because I did not have time to maintain it.

I was able to dig up the APK file, so if you are using an Android device you might still be able to install it: trash-belt.apk.
